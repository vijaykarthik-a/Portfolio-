# Portfolio Website

A modern, responsive portfolio website built with React.

## Features

- Smooth scrolling navigation
- Animated hero section
- Projects showcase
- Skills visualization
- Contact form
- Fully responsive design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

- Update your name in `src/components/Hero.js`
- Add your projects in `src/components/Projects.js`
- Update skills in `src/components/Skills.js`
- Change social links in `src/components/Footer.js`
- Customize colors in the CSS files

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
